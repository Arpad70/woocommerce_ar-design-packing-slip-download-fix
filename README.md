# Ar Design Packing Slip Download Fix

Malý samostatný WordPress plugin pre ar-design.sk.

## Čo robí
- dopĺňa tlačidlo **Stáhnout dodací list** do sekcie dodacieho listu v detaile WooCommerce objednávky,
- zobrazí tlačidlo iba vtedy, keď už je dokument `packing-slip` vygenerovaný,
- využíva natívne API pluginu WooCommerce PDF Invoices & Packing Slips,
- podporuje aktualizácie z GitHub Releases priamo vo WordPresse.

## Závislosti
- WooCommerce
- WooCommerce PDF Invoices & Packing Slips

## GitHub release workflow
Aby fungovali aktualizácie vo WordPresse, publikovaný release musí obsahovať ZIP asset s názvom:

- `ar-design-packing-slip-download-fix.zip`

Repozitár očakávaný v hlavičke pluginu:

- `Arpad70/woocommerce_ar-design-packing-slip-download-fix`
